<h3>Usage</h3>

1. First Clonning
```bash
git clone https://github.com/Brspontes/next-level.git
```

2. In Terminal usage npm
```bash
npm install
```
3. Execute te application npm | yarn
```bash
npm run dev
```
or
```bash
yarn dev
```
